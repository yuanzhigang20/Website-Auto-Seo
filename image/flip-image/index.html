FAQ

Is this tool free?

Yes. It is free and does not require an account.

Does it work offline?

After the page loads, the processing itself is local. Browser refreshes still need the website assets unless cached.

Which image formats work?

Most modern browsers can open JPG, PNG, WebP, GIF still frames, BMP, and SVG images through the built-in image decoder.
