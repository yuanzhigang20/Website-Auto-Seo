FAQ

What is a noindex tag?

A noindex tag tells search engines not to include a page in search results.

How do I check if a page has noindex?

Inspect the HTML meta robots tag or HTTP X-Robots-Tag header for a noindex directive.

What is X-Robots-Tag?

X-Robots-Tag is an HTTP header that can apply indexing directives such as noindex outside the HTML body.

Is noindex the same as robots.txt disallow?

No. Robots.txt controls crawling, while noindex controls whether a crawled page can appear in search results.

Can noindex remove a page from Google?

Yes, after Google recrawls the page and sees the noindex directive.

Should noindex pages be in a sitemap?

Usually no. Sitemaps should focus on canonical pages you want indexed.
