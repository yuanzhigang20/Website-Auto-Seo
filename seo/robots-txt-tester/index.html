FAQ

How do I test a robots.txt file online?

Paste the robots.txt rules into the tool, add the URL or path you want to verify, and review whether the rules allow or disallow crawling.

What is the difference between a robots.txt tester and checker?

A tester focuses on trying a URL against rules, while a checker also reviews directives, sitemap hints, and common formatting problems.

Can this tool check if a URL is blocked?

Yes. Use it to review whether a path appears to be blocked by matching Disallow or Allow directives.

How do Allow and Disallow rules work?

Disallow tells crawlers which paths not to crawl, while Allow can reopen a more specific path inside a blocked section.

Can robots.txt remove a page from Google?

No. Robots.txt controls crawling, not indexing. Use noindex for pages you want removed from search results.
